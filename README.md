# To-Do List Application (C++)

## Overview
This project is a console-based To-Do List Application developed using C++. It was created as part of my C++ Programming Internship.

The application helps users manage their daily tasks by allowing them to add tasks, view tasks, and mark tasks as completed.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Persistent task storage using file handling

## Technologies Used
- C++
- Object-Oriented Programming
- Vectors (STL)
- File Handling

## How It Works
The program stores tasks using a vector of Task objects. Each task contains a description and a completion status. Tasks are saved to a text file so that they remain available even after the program is closed.

## Concepts Demonstrated
- Classes and Objects
- Dynamic data storage using vectors
- File input/output operations
- Menu-driven console applications

## How to Run

Compile the program:

g++ main.cpp -o todo

Run the program:

./todo
